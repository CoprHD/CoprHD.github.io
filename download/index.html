Download CoprHD

There are a few ways to get CoprHD: download an appliance, use Vagrant, or build from source. For beginning users and developers, we recommend you start with the OVF or vagrant setup.

The most recent release of CoprHD is 2.4.

Download an appliance

Appliances are brand new - please report issues on the Google Group or Jira.

  • CoprHD 2.4 OVF
  • CoprHD 2.4 VMDK disk

Vagrant Setup

Curt Bruns has created an easy to use Vagrant setup which includes ScaleIO for for a complete storage stack on a laptop. And if you want to get really fancy, he has another branch that includes DevStack.

  • CoprHD 2.4 + ScaleIO
  • CoprHD 2.4 + ScaleIO + DevStack

Build from source

Recommended for more experienced users. Right now it's only confirmed to work on OpenSuSE, although there are reports in the wild of running CoprHD on Ubuntu.
