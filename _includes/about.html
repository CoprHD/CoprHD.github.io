About CoprHD

CoprHD is an open software defined storage controller platform based on EMC's ViPR Controller. It enables management and automation of storage resources for block and file storage platforms.

  • Enables management and automation of storage resources for block and file storage platforms.
  • Centralizes and transforms multi-vendor storage.
  • Abstracts resources to deliver automated, policy-driven storage services on demand via a self-service catalog.
